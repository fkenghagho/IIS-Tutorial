Part 1: Determine the constant c

For f(x) to be a valid PDF, the total area under the curve must equal 1.

Integral of f(x) from -∞ to +∞ = 1

∫-∞+∞ f(x) dx = 1

Since f(x) is non-zero only for 4 ≤ x ≤ 6:

∫46 c * x4 dx = 1

c * ∫46 x4 dx = 1

c = 5 / 6752

The constant c is: c = 5 / 6752
