Kalman Filter & Localization Analysis

1. The Kalman Filter Framework

The Kalman filter tracks the state of a system using a prediction-correction loop. The mathematical core consists of the state transition and measurement models:

State Transition Equation:

$$X_t = A_t X_{t-1} + B u_t + Q$$

Determines the next state $X_t$ based on the previous state and control inputs.

Measurement Equation:

$$Y_t = C_t X_t + R$$

Maps the true state $X_t$ to the observed sensor values $Y_t$.

2. Linear Motion Models

Constant Acceleration

Assuming acceleration $a$ is constant over the total duration $t$:

$$x_t = \frac{1}{2}at^2 + v_0t + x_0$$

$$v_t = at + v_0$$

Variable Acceleration

Assuming acceleration is nearly constant within a small interval $\Delta t$:

$$x_t = x_{t-1} + v_{t-1}\Delta t + \frac{1}{2}a\Delta t^2$$

$$v_t = v_{t-1} + a\Delta t$$
